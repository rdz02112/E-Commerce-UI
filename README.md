# UI — E-Commerce Dashboard

## 🚀 Quick Start

```bash
npm install
npm run dev
```

## 📦 Stack
- React 18 + Vite
- Recharts (charts)
- Pure CSS (no Tailwind needed)

## 🗂 Structure

```
src/
├── App.jsx              # Router
├── data.js              # All mock data
├── index.css            # Global styles + design tokens
├── components/
│   ├── Sidebar.jsx
│   └── Topbar.jsx
└── pages/
    ├── Dashboard.jsx    # Overview + charts
    ├── Products.jsx     # Product table + filter
    ├── Orders.jsx       # Order table + status filter
    └── Customers.jsx    # Customer table + sort
```
